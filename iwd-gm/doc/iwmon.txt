IWMON(1)
========
:doctype: manpage


NAME
----
iwmon - Wireless monitor


SYNOPSIS
--------
*iwmon* ['OPTIONS']


DESCRIPTION
-----------
The iwmon(1) command provides access to the wireless subsystem commands,
responses and events from the nl80211 generic netlink interface.
